MPE+ Tool

MPE+ Tool

Mobile Phone Examiner Plus (MPE+) is a mobile device data review tool that can be used in the field as part of a mobile field unit or in the lab. The data extracted from mobile devices, using MPE+, can be imported into an FTK case.
Supplier access data
Supplier website https://accessdata.com/

Key Features of MPE+

  • Enhanced smart device acquision and analysis capabilities
  • Support for IOS,Android,Blackberry,Windows phone
  • ability to acquire data from the e internal memory of supported mobile devices and associated media

MPE+ is tool designed to ease mobile device forensic process.It can perform various forensic data acquision tasks.It can retrive data such as equipment/user data, PIM Data, Call Logs, SMS Messages, MMS Messages, Stand-alone Files, Application data,social media data,internet data, GPS data, Non-Latin Characters, Hashing, Case File Data Protection.

Once data is imported into FTK case, it offers more in-depth drill-down, categorization, full-text index searching, and all of this is right alongside other digital evidence collected for a case.
